## ⚙️ Installation

```bash
git clone https://github.com/Ushapithani/gemini-chat-clone.git
cd gemini-chat-clone
npm install
````

Create `.env` file:

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```

Run project:

```bash
npm run dev
```
