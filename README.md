It replicates a real AI chat experience with a clean UI and smooth interactions.

## ✨ Features
- Google Gemini API integration  
- Chat UI with responses  
- Dark mode  
- Sidebar collapse/expand  
- Typing animation  
- Chat history  
- Responsive design  
- Context API state management  
- Vercel deployment
